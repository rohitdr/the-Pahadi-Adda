# Modern Restaurant Landing Page Template (React + Tailwind)

A sleek, modern, and fully responsive restaurant landing page built with **React + Vite + Tailwind CSS + Framer Motion**.

Perfect for developers, freelancers, or restaurant owners who want a beautiful website without starting from scratch.


## 🚀 Live Demo

👉 https://zaika-kitchen-xi.vercel.app/



## ✨ Features

* ⚡ Modern and clean UI design
* 📱 Fully responsive (mobile-first)
* 🎬 Smooth animations using Framer Motion
* 🍔 Sections included:

  * Hero Section
  * Navbar
  * Services / Menu Preview
  * About Section
  * Testimonials
  * Gallery
  * Call-To-Action (CTA)
  * Footer
* 🎨 Easy to customize (text, images, colors)
* 🧩 Reusable components



## 🛠️ Tech Stack

* React (Vite)
* Tailwind CSS
* Framer Motion
* React Icons



## 📦 Installation & Setup

Clone the repository:

```bash
git clone https://github.com/rohitdr/ZaikaKitchen.git
```

Navigate to project folder:

```bash
cd ZaikaKitchen
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```



## 🎯 Customization Guide

### ✏️ Change Text Content

* Go to `src/components/`
* Edit text directly inside components



### 🖼️ Change Images

* Replace image URLs in components
* Or add your images inside `/public` and update paths



### 🎨 Change Colors

* Edit Tailwind classes like:

```bash
bg-black text-white
```

* Or update Tailwind config for theme colors
